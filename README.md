# 📱 Anurag Mobile Repairing Centre

**Anurag Mobile Repairing Centre** is a professional mobile phone repairing shop website that helps customers explore mobile features, search models by brand, and easily contact the shop for fast and reliable repair services.

🌐 **Live Website:** https://anuragmobile.in

---

## 🛠️ Services We Provide

- 📱 Mobile Screen Replacement  
- 🔋 Battery Replacement  
- 🔌 Charging Port Repair  
- 📷 Camera Repair  
- 🧠 Motherboard Repair  
- 🧩 Genuine Spare Parts  
- ⚡ Fast Diagnosis & Repair
-     Track Your Device Progress  

We repair **all major smartphone brands**, including old and new models.

---

## 🔍 Website Features

- ✅ Brand-wise mobile selection (Apple, Samsung, Redmi, OnePlus, Oppo, Vivo, etc.)
- ✅ Search mobile models with full specifications
- ✅ SEO-friendly pages for better Google ranking
- ✅ Fully responsive design (Mobile / Tablet / Desktop)
- ✅ Fast performance using Vite
- ✅ Custom domain support
- ✅ Clean UI & smooth navigation

---

## 🧑‍💻 Tech Stack

- **Frontend:** React + TypeScript  
- **Build Tool:** Vite  
- **Styling:** Tailwind CSS  
- **Icons:** Lucide React  
- **Hosting:** GitHub Pages (Custom Domain)  
- **Version Control:** Git & GitHub
